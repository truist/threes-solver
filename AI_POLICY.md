## AI usage disclosure

The code in this repo/crate was developed with assistance from AI / LLM tools.

In general, AI was used for learning purposes, not to generate code.
(This project was an exercise in learning Rust; using AI to generate the code would have been counter-productive.)

The tee_output crate is an exception.
It was generated entirely by AI.
Those sources are marked with `AI-ORIGIN` comments.

In at least one instance AI was used to move existing code around between directories.

All code was reviewed, edited, and tested by the maintainer.

## AI contribution policy

AI tools cannot take responsibility for code; humans have to do that.
You have to do that.

All contributions must be reviewed, edited, and tested by the contributor.

Any contributions generated by AI must be marked with `AI-ORIGIN` comments.
